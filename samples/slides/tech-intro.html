# Your browser is the IDE Zero install. Zero setup. Just open and build. --- ## The Problem - Modern IDEs are heavy — VS Code, WebStorm, Figma... - Installation takes time and breaks across machines - Sharing work requires accounts, plugins, or exports - Creative flow stops the moment setup begins --- ## The Solution **Single HTML files. Nothing to install.** - Open in any browser — Chrome, Firefox, Safari - Works offline, on any OS, on any device - Share by sending one file - No accounts. No extensions. No build step. --- ## designer.html A visual HTML layout editor, right in your browser. - Drag-and-drop canvas with live preview - Insert text, images, shapes, and containers - Export clean HTML with one click - Grid snapping and alignment guides built in --- ## slides.html A Markdown & HTML slide editor with instant preview. - Write slides in plain Markdown or raw HTML - Split slides with `---` — no special syntax - Live side-by-side editor and preview - Present directly from the browser — fullscreen ready **Get started:** open `designer.html` or `slides.html` in your browser.
